# cooking-cookie
수제쿠키 만들기

## 수제 붕어빵 만들기

1. 붕어빵 믹스와 물을 1:1.4의 비율로 반죽합니다. 
*붕어빵믹스 링크* : http://www.cookiebebe.com/shop/shopdetail.html?branduid=687
2. 붕어빵 틀 모양의 팬을 준비하여 버터 또는 식용유로 기름칠을 합니다.
3. 약불로 앞뒤로 3분씩 팬을 뒤집으며

## 수제 아이스크림 만들기

1. 편의점을 간다
2. 아이스크림을 산다
4. 맛있게 먹고싶지만 기다립니다
5. 붕어빵믹스를 넣는다
6. 붕어빵 안에 넣을 팥소를 준비합니다.
